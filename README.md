#WeatherByYp

Weather application that displays real time information including humidity level, description, temperature and more from OpenWeatherMap API.

#Technologies used
-HTML
-CSS
-JavaScript
-OpenWeatherMap API

#Features
- Search weather by city name
- Displays temperature in °C
- Shows humidity percentage
- Weather description
- Weather emoji based on conditions
